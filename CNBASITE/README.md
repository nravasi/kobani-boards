# CNBASITE — CABNA Club Website Project

Content, design tokens, and scraped data for the Club Atlético Banco de la Nación Argentina (CABNA) website rebuild. This project contains a Python scraper that pulls structured content from [clubbanconacion.org.ar](https://clubbanconacion.org.ar/) and the [@cabnaoficial](https://www.instagram.com/cabnaoficial/) Instagram profile, plus three complete visual style definitions ready for front-end implementation.

Install the following tools before you begin. The versions listed are the minimum supported.

| Tool | Minimum Version | Check Command |
|------|----------------|---------------|
| [Git](https://git-scm.com/) | 2.30+ | `git --version` |

## Local Setup

### 1. Clone the repository

```bash
git clone https://github.com/nravasi/kobani-boards.git
cd kobani-boards/CNBASITE
```

### 2. Create a virtual environment (recommended)

A virtual environment keeps project dependencies isolated from your system Python.

**macOS / Linux:**

```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows (Command Prompt):**

```cmd
python -m venv venv
venv\Scripts\activate
```

**Windows (PowerShell):**

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

### 3. Install dependencies

```bash
pip install requests beautifulsoup4
```

This installs two packages:

- **requests** — HTTP library used to fetch web pages.
- **beautifulsoup4** — HTML parser used to extract structured content.

### 4. Run the scraper

```bash
python3 scraper.py
```

On Windows, use `python scraper.py` instead.

The scraper fetches content from the live CABNA website and Instagram profile. It writes four JSON files into the `data/` directory. A full run takes roughly 30–60 seconds depending on network speed.

### 5. Run the tests

```bash
python3 -m unittest test_scraper -v
```

Tests validate the scraper's parsing logic using fixtures. They do not make network requests.

### 6. Deactivate the virtual environment (when finished)

```bash
deactivate
```

## Styles

The project defines three distinct visual themes. Each theme provides a complete design language — colour palette, typography, component treatment, and layout rules. Detailed specifications live in `style-guide.md`. Machine-readable tokens for each theme live in their corresponding `tokens-style*.json` file.

### Style 1: Estilo Puro — Modern Minimal

**Token file:** `tokens-style1.json`

**Aesthetic intent:** Clean, calm, and contemporary. The interface is stripped to essentials so the club's content takes centre stage. A light page background (`#FAFAFA`) with deep navy headings (`#1A1A2E`) and coral-red accents (`#E94560`) creates a polished, approachable feel. A single sans-serif typeface (Inter) for both headings and body keeps things unified. Rounded corners (0.75 rem) and soft shadows give cards a friendly lift. This style suits a broad audience and signals digital maturity.

**Visual summary:**

- **Background:** Light grey (`#FAFAFA`)
- **Primary colour:** Deep navy `#1A1A2E`
- **Accent colour:** Coral `#E94560`
- **Fonts:** Inter for headings and body
- **Cards:** White, bordered, with soft shadows and rounded corners
- **Buttons:** Coral fill, sentence case, 0.5 rem radius
- **Hero height:** 70 vh

### Style 2: Cancha Viva — Bold Athletic

**Token file:** `tokens-style2.json`

**Aesthetic intent:** Electric and high-energy. A near-black background (`#0D0D0D`) lets vivid red (`#FF2D55`), yellow (`#FFB800`), and teal (`#00D4AA`) colours punch through like stadium lights. Oversized uppercase headings in Bebas Neue evoke scoreboard typography. Sharp corners (0.25 rem) and red accent bars give components a broadcast-graphics look. This style targets a younger, match-day-focused audience that expects bold visuals.

**Visual summary:**

- **Background:** Near-black (`#0D0D0D`)
- **Primary colour:** Vivid red `#FF2D55`
- **Accent colour:** Electric yellow `#FFB800`
- **Fonts:** Bebas Neue for headings, Roboto for body
- **Cards:** Dark surface, 3 px red accent bar at top, glowing hover state
- **Buttons:** Red fill, uppercase, tight radius with hover glow
- **Hero height:** 90 vh

### Style 3: Tribuna Dorada — Classic Prestigious

**Token file:** `tokens-style3.json`

**Aesthetic intent:** Elegant, warm, and authoritative. Deep teal (`#1B3A4B`) conveys trust, while gold accents (`#C8A951`) add prestige tied to the club's heritage. Playfair Display, a transitional serif, gives headings an editorial quality. Warm cream backgrounds (`#FAF8F4`) replace cold whites. This style suits a membership-focused audience that values history and tradition.

**Visual summary:**

- **Background:** Warm cream (`#FAF8F4`)
- **Primary colour:** Deep teal `#1B3A4B`
- **Accent colour:** Gold `#C8A951`
- **Fonts:** Playfair Display for headings, Source Sans 3 for body
- **Cards:** White on cream, gold top-border on featured cards, editorial spacing
- **Buttons:** Teal fill (with a gold variant for premium CTAs), sentence case
- **Hero height:** 75 vh

### Switching Between Styles

Each token file can be consumed by a build tool to generate CSS custom properties. To switch themes at runtime:

1. **Generate CSS variables** from each `tokens-style*.json` file (manually or via a build script).
2. **Scope each set of variables** under a class on the `<html>` element:

```css
/* From tokens-style1.json */
.theme-estilo-puro {
  --color-primary: #1A1A2E;
  --color-secondary: #E94560;
  --color-background: #FAFAFA;
  --font-heading: 'Inter', 'Helvetica Neue', Arial, sans-serif;
  /* ... */
}

/* From tokens-style2.json */
.theme-cancha-viva {
  --color-primary: #FF2D55;
  --color-secondary: #FFB800;
  --color-background: #0D0D0D;
  --font-heading: 'Bebas Neue', 'Oswald', 'Impact', sans-serif;
  /* ... */
}

/* From tokens-style3.json */
.theme-tribuna-dorada {
  --color-primary: #1B3A4B;
  --color-secondary: #C8A951;
  --color-background: #FAF8F4;
  --font-heading: 'Playfair Display', 'Georgia', 'Times New Roman', serif;
  /* ... */
}
```

3. **Apply the desired theme** by setting the class on `<html>`:

```html
<html class="theme-estilo-puro">
```

4. **To switch**, replace the class name:

```js
document.documentElement.className = 'theme-cancha-viva';
```

5. **Load the required fonts** for the active theme via Google Fonts:

| Theme | Google Fonts Link |
|-------|-------------------|
| Estilo Puro | `https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap` |
| Cancha Viva | `https://fonts.googleapis.com/css2?family=Bebas+Neue&family=Roboto:wght@300;400;500;700&display=swap` |
| Tribuna Dorada | `https://fonts.googleapis.com/css2?family=Playfair+Display:wght@400;600;700&family=Source+Sans+3:wght@300;400;600;700&display=swap` |

## Troubleshooting

| Problem | Solution |
|---------|----------|
| `python3: command not found` (Windows) | Use `python` or `py -3` instead of `python3`. |
| `pip: command not found` | Try `pip3`, or run `python3 -m pip install ...`. |
| `ModuleNotFoundError: No module named 'requests'` | Run `pip install requests beautifulsoup4` (activate your venv first). |
| Scraper times out or returns empty data | The live CABNA website may be down. Pre-scraped data is already committed in `data/`. |
| Tests fail with import errors | Ensure you are running tests from the `CNBASITE/` directory. |
| `venv\Scripts\Activate.ps1` blocked on Windows | Run `Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned` in PowerShell, then try again. |
